# 汽车电子七部曲

一个持续更新的开源技术书系列——从计算的本质讲到底层物理、从逻辑门讲到整车通信、从裸机讲到工程精神，再到诊断、安全、存储三大专题的深度展开。

## 如何阅读

- **[从沙子到车辙——一个工程师的理解](from-sand-to-ruts/README.md)** 是全系列的**技术总纲**，建立从计算本质到汽车电子的全局认知，建议先读。
- 之后可按需深入专题分册：**PTP**（时间同步）、**HSM**（硬件安全模块）、**存储**（文件系统）、**UDS**（诊断协议）。每本专题书都从思想实验/元问题出发，逐机制拆解协议或规范，再走进开源工业级源码，最后亲手实现一个教学级的轻量版本。
- 每本书内部按章节编号（如 `1.1`、`3.2`）顺序阅读即可，左侧目录已按章节分组，点击左上角搜索图标可全文检索，点击右上角图标切换深色模式。
